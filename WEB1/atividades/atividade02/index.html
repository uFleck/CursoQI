Tags Descontinuadas no HTML5

  • <applet> substituída por <object>
  • <basefont> substituída por estilos de fonte CSS
  • <center> substituída por estilo CSS. Ex:” text-align:Center”
  • <dir> substituída por <ul>
  • <u> substituída por estilos CSS
  • <isindex> substituída por <form>
